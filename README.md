# Klondike
A Klondike game written in C++
![Exemple](resource/exemple.png)

# Installation

You will need the SFML library :
```
# on Ubuntu

sudo apt-get install libsfml-dev

# on Mac OS

brew install sfml
```

# Build

After cloning the repository enter the following commands to build the game :
```
make
```

# Usage
```
./Klondike
```
