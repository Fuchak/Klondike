# Klondike
A Klondike game written in C++
![Exemple](resource/exemple.png)

# Build
You will need thoses dependencies :
```
cmake (3.10 minimum)
pthread
opengl
xlib
xrandr
freetype
glew
jpeg
sndfile
openal
```
After cloning the repository enter the following commands to build the game :
```
mkdir build
cd build
cmake ..
cmake --build .
```

# Usage
```
./Klondike
```
